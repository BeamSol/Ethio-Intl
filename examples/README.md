# Examples

This directory contains example implementations and demonstrations of the Ethio-Intl library.

## Contents

- `demo.html` - Interactive web demo showcasing all library features
- `demo-utils.js` - Utility functions for the demo
- `demo-simple.html` - Simplified demo example
- `src-examples/` - Source code examples for different use cases

## Running Examples

### Web Demo
```bash
# Start a local server
python -m http.server 5500

# Open in browser
http://localhost:5500/examples/demo.html
```

## Features Demonstrated

- **Amharic Transliteration**: Real-time typing conversion
- **Ethiopian Calendar**: Gregorian to Ethiopian date conversion
- **Geez Numerals**: Arabic to Geez numeral conversion
- **Internationalization**: Multi-language support with React context

See the source examples in `src-examples/` for integration patterns.
